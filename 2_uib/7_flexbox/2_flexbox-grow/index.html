Flexbox Advanced

flex-grow: ;

mit flex-grow: ; können wir definieren, wieviel platz ein element innerhalb des elternelementes einnimmt, ohne pixelangaben, oder ähnliche größenangaben.

der default wert von flex-grow: ; ist 0, das bedeutet, das die elemente nur soviel platz einnehmen, wie sie brauchen um ihren inhalt anzuzeigen.
