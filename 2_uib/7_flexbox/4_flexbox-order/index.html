order: ;

mit order: ; können wir die angezeigte reihenfolge von elementen verändern, egal wie sie im HTML code sortiert sind.

der standardwert von order: ; ist 0.

1 (0)
2 (-1)
3 (3)
4 (1)
5 (-2)

5 (-2)
2 (-1)
1 (0)
4 (1)
3 (3)

Wenn elemente die gleiche order: ; haben, werden sie in der reihenfolge angezeigt, in der sim HTML code stehen.

1 (0)
2 (-1)
3 (3)
4 (2)
5 (-2)
BOX (-1)
