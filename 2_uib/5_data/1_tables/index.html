Tabellen

Default Table

Eine Tabelle erstellen wir mit dem element "table", dort können wir dann "tr" für eine neue zeile (table row) und dort drin dann "td" (table data) für eine neue spalte nutzen.

Tabellen Header

Um einer tabelle eine kopfzeile zu geben, nutzen wir das "th" element, innerhalb einer eigenen zeile (tr).

Fluide Tabellen & Tabellen Titel

Wenn wir die größe von spalten ändern ist der inhalt der "th", also des table headers zentriert.

mit "caption" können wir der tabelle einen titel geben, das caption element sollte immer das erste kind von "table" sein, also über "tr".
